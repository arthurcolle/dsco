ARCHITECTURE
================================================================================

Main Flow:
  1. CLI dispatch (main.c) parses args → selects mode
  2. Mode: oneshot query, interactive REPL, or topology list
  3. Invokes llm_stream_request() to stream Claude API
  4. LLM returns tool calls → tool_dispatch() executes them
  5. Tools can spawn swarm agents (fork + IPC)
  6. Results flow back to LLM for next iteration

Swarm Hierarchy:
  • Parent spawns child agents via fork()
  • Each child is independent dsco process
  • IPC channels for messages, shared scratchpad
  • Up to 4 levels deep (configurable)
